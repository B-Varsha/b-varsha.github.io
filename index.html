Projects

Airline Reservation system
This project facilitates seamless airline reservations through real-time data streaming via Apache Kafka. Uses the analytical power of Apache Spark and Spark SQL to extract valuable insights from reservation data. Enhances efficiency and precision in managing airline bookings for an optimized customer experience.
Immunization Effectiveness in Rural areas
This project employs machine learning regression models to analyze demographic, clinical, and genomic data, predicting childhood immunization likelihood among Indian women. It addresses regional variations, including urban, hilly, and rural areas, aiming to enhance vaccine coverage and personalize strategies for vulnerable populations.
PennyWise-Expense Management System
PennyWise is a web application streamlining bill splitting. With a user-friendly interface, it allows easy receipt uploading, item selection, and precise cost calculation, offering an efficient tool for groups managing shared expenses.
VoidVoices
This web application, designed for enhanced accessibility for differently-abled individuals, employs deep learning (OpenCV) for sign language to text conversion and JavaScript for Braille conversion. It addresses various aspects of disability, providing a user-friendly platform for improved communication and inclusivity.
Multi-Terrain Hexapod Design
The project involves crafting a multi-terrain navigating hexapod with an integrated robotic arm. Leveraging deep neural networks, it enables environment monitoring, object identification, and selective removal through the robotic arm based on specific criteria, enhancing intelligent and adaptable robotic functionalities.
Background Removal using PCA
In this project,Principal Component Analysis (PCA) is used as a dimensionality reduction tool isolating foreground from background. By extracting essential components, PCA enhances clarity, focusing on main elements and reducing background noise for improved visual quality.
